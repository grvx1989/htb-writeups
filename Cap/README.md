# Cap

## Machine Overview
Machine focused on web enumeration, traffic analysis, and privilege escalation through insecure file access.

## Key Concepts
1. Web Enumeration (HTTP service discovery)
2. PCAP Analysis (network traffic inspection)
3. IDOR (Insecure Direct Object Reference)
4. Credential Extraction
5. Privilege Escalation (abusing Linux capabilities)

## Tools Used
1. Nmap
2. Browser
3. Wireshark / tcpdump
4. Linux CLI

## Summary
Gained access by analyzing network capture files exposed through a web application, extracting credentials, and leveraging misconfigured capabilities to escalate privileges.
