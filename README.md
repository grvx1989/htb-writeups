# Hack The Box writeups
Hands-on Hack The Box labs focused on enumeration, explotation and privilege escalation.


## Methodology
1. Enumeration (Nmap, service discovery)
2. Service Analysis (version identification)
3. Exploitation (known vulnerabilities / manual techniques)
4. Privilege Escalation
5. Documentation

## Tools Used
1. Nmap
2. Netcat
3. Telnet
4. Linux CLI

## Machines completed
-Meow

## Goal
To build solid foundations in cybersecurity through practical labs and real scenarios.
