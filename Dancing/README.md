
#Dancing

## Machine Overview
Machine focused on SMB enumeration and accessing shared resources through misconfigured permissions.

## Key Concepts
1. SMB Enumeration (service discovery on ports 139/445)
2. Anonymous Access (null sessions)
3. Shared Folder Exploration
4. File Retrieval
5. Credential Exposure

## Tools Used
1. Nmap
2. smbclient
3. Linux CLI
4. Summary

Gained access by enumerating SMB shares with anonymous login, navigating exposed directories, and retrieving sensitive files from misconfigured network shares.
