SMB Enumeration (service discovery on ports 139/445)

Shared Folder Exploration
File Retrieval
Credential Exposure

Nmap
smbclient
Linux CLI
Summary

Gained access by enumerating SMB shares with anonymous login, navigating exposed directories, and retrieving sensitive files from misconfigured network shares.
