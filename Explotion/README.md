# Explosion

## Methodology
1. Enumeration (Port scanning)
2. Service Identification (RDP)
3. Authentication (weak / no credentials)
4. Remote Access

## Tools Used
1. Nmap
2. RDP Client

## Key Concepts
1. Open RDP service
2. Weak authentication / misconfiguration
3. Remote desktop access
4. Initial foothold via RDP

## Summary
Explosion focuses on gaining access through an exposed RDP service with weak or misconfigured authentication. 
By identifying the open port and connecting via RDP, it is possible to access the system and retrieve the flag.
