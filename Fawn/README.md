# Fawn 

## Overview
Introductory machine focused on basic service enumeration and remote access.

## Methodology
1. Enumeration (Nmap scan)
2. Service Identification (FTP - port 21)
3. Exploitation (Anonymous login)
4. Flag Retrieval

## Tools Used
- Nmap
- FTP
- Linux CLI

## Key Finding
FTP service allowed anonymous login, leading to direct access and flag retrieval.

## Goal
Understand basic enumeration and how misconfigured services can lead to compromise.
