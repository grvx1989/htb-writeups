# Crocodile

## Methodology
1. Enumeration (Nmap, descubrimiento de servicios)
2. Service Analysis (FTP anonymous / credenciales débiles)
3. Exploitation (login FTP + acceso a archivos)
4. Data Extraction (búsqueda de credenciales / información útil)

## Key Concepts

1. FTP Authentication
2. Anonymous Login / Weak Credentials
3. File Enumeration
4. Credential Reuse

## Tools Used
1. Nmap
2. FTP
3. Hydra (opcional para fuerza bruta)

## Summary

Se identificó el servicio FTP expuesto.
Mediante credenciales débiles (o acceso permitido), se logró autenticación y enumeración de archivos, encontrando información sensible utilizada para avanzar en el laboratorio.
